What is HTML action Attribute?

HTML, HTML Attributes HTML, HTML AttributesSeptember 24, 2019Rajnish Rajput 0 Comments

The action attribute indicates where to send the form data on submitting the form by the user. (e.g. a PHP script that handles the form data).

You can use action attribute on form tag.
