]]> The class attribute is very important as it is a part of global attribute in HTML and used to allow CSS and JavaScript to select the specific HTML element.

In CSS you can use CSS class selectors dot (.) and DOM method in JavaScript like: document.getElementsByClassName.

Tag Support

You can use class attribute on all the elements in HTML.

Example

<div class="tagcloud">
    <a href="https://buginit.com/tag/html/" class="tag-cloud-link">
        HTML
    </a>
    <a href="https://buginit.com/tag/html-attributes/" class="tag-cloud-link">
        HTML Attributes
    </a>
    <a href="https://buginit.com/tag/html-global-attributes/" class="tag-cloud-link">
        HTML Global Attributes
    </a>
</div>
<p class="important">Note that this is an important paragraph. :)</p>

Note: you can also use multiple classes

    </a>
    <a href="https://buginit.com/tag/html-attributes/" class="tag-cloud-link tag-link-6 tag-link-position-2">
        HTML Attributes
    </a>
    <a href="https://buginit.com/tag/html-global-attributes/" class="tag-cloud-link tag-link-7 tag-link-position-3">

    </a>
</div>
<p class="important bold">Note that this is an important paragraph. :) and also bold</p>

]]> Global accesskey attribute provides a hint for the current element to generate a keyboard shortcut.

The value of the attribute must be a single printable character (including accented and other characters that the keyboard can generate)

In separate browsers, the manner to access the shortcut key varies.

As this is the part of HTML global attributes you can use this on any HTML tag.

Browser Windows Linux Mac
Internet Explorer [Alt] + accesskey N/A
Chrome [Alt] + accesskey [Alt] + accesskey [Control] [Alt] + accesskey
Firefox [Alt] [Shift] + accesskey [Alt] [Shift] + accesskey [Control] [Alt] + accesskey
Safari [Alt] + accesskey N/A [Control] [Alt] + accesskey
Opera Opera 15 or newer: [Alt] + accesskey
Opera 12.1 or older: [Shift] [Esc] + accesskey

However, the shortcut can be put to another key mix in most browsers.

Example

<p>If you want to save the 
  <strong><u>S</u></strong>ettings 
  press the below button
</p>

<button accesskey="s">Save</button>
<button accesskey="c">Cancel</button>

Note: In the WHATWG spec, it says you can specify multiple space-separated characters, and the browser will use the first one it supports. However, this does not work in most browsers. IE/Edge uses the first one it supports without problems, provided there are no conflicts with other commands.
